## Internal Linking Map

Key pages that should link to each other for SEO:

- Buyer guide → Link to relevant comparison (e.g., drills guide → DeWalt vs Milwaukee)
- How-to → Link to relevant buyer guide (e.g., "how to tile" → "best tile saws")
- Review → Link to parent buyer guide (e.g., DeWalt review → "best drills" guide)
- Comparison → Link to both brand's product pages
